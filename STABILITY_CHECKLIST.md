# 안정화 체크리스트 (현재 상태)

목적: 기능/다국어/SEO 상태를 빠르게 점검한다.  
범위: 코드 변경 없이 확인 가능한 항목 위주.

## 1) 기능 안정화
- 홈 화면 로딩 (히어로/검색/카드)
- 시술 목록/병원 목록 페이지 렌더
- 시술 상세/병원 상세 페이지 정상 렌더
- 문의 페이지 폼/토스트/업로드 흐름 정상
- 콘솔 에러 없음

## 2) 다국어 준비 상태
- 주요 문구는 `t()` 기반으로 분리됨
- 가격/날짜 포맷은 유틸(`format*`)로 통합됨
- 리스트 페이지 타이틀/메타도 키 기반으로 연결됨

## 3) SEO 점검
- `sitemap.xml` 생성 경로 확인
- `robots.txt` 정책 확인
- 상세 페이지 JSON-LD 정상 출력
- 리스트 페이지 canonical 지정됨 (`/treatments`, `/hospitals`)

## 4) 확인 방법 (간단)
- 브라우저에서 `/`, `/treatments`, `/hospitals`, `/treatments/{slug}`, `/hospitals/{slug}` 순서로 확인
- 콘솔에 오류 없는지 확인
